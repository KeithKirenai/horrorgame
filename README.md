# Echo Index — Godot Horror Game

First-person horror prototype built in Godot 4. Procedural industrial maze, generator-powered lighting, entity stalker AI.

## Controls
- **WASD**: Move
- **Mouse**: Look
- **E**: Interact / Crank Generator
- **F**: Flashlight
- **Shift**: Sprint

## How to Run
Open in Godot 4.3+ and run (`main.tscn`).
