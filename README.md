# PSX Horror Prototype (Godot Version)

This is a Godot 4 port of the PSX Horror Prototype.

## Controls
- **WASD**: Move
- **Mouse**: Look
- **Click** (on start screen): Start Game / Capture Mouse
- **ESC**: Release Mouse

## Features
- **PSX Aesthetic**: 320x240 internal resolution, upscaled with dithering, color quantization (banding), and vertex wobble (simulated via low res).
- **Atmosphere**: Deep fog, dark ambient, flashlight.
- **Procedural Generation**: Random debris placement.

## How to Run
Open this folder in Godot 4.3+ and run the project (F5).
The main scene is `main.tscn`.
